
The package `SemiAlgebraicTypes.jl` provides implementation of semialgebraic sets such as

- [points, lines, spheres, cones, cylinders, ellipsoids, ...](@ref basic)
- [meshes](@ref mesh)
- [bspline curves, surfaces, volumes](@ref splines)
